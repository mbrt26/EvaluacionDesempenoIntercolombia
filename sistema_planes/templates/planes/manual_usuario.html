1. Introducción

¿Qué es el Sistema de Planes de Mejoramiento?

El Sistema de Planes de Mejoramiento es una plataforma web diseñada para gestionar el ciclo completo de planes de mejoramiento para proveedores que obtienen evaluaciones de desempeño inferiores a 80 puntos.

Objetivo del Sistema
  • Automatizar el proceso de gestión de planes de mejoramiento
  • Facilitar la comunicación entre proveedores y la organización
  • Mantener trazabilidad completa de todos los cambios
  • Asegurar el cumplimiento de plazos y requisitos
  • Generar reportes y estadísticas de gestión
Importante: Según la parametrización ITCO-ISA, solo las evaluaciones con puntaje inferior a 80 puntos requieren un plan de mejoramiento.

2. Roles del Sistema

Proveedor

Responsabilidades:

  • Crear planes de mejoramiento
  • Firmar y enviar documentos
  • Responder aclaraciones
  • Realizar ajustes solicitados
Técnico

Responsabilidades:

  • Crear evaluaciones
  • Enviar cartas de evaluación
  • Revisar planes
  • Aprobar o solicitar ajustes
Gestor

Responsabilidades:

  • Supervisar todo el proceso
  • Aprobar planes para radicación
  • Solicitar aclaraciones
  • Marcar faltas de ética
  • Generar reportes
Gestor de Compras

Responsabilidades:

  • Radicar planes aprobados
  • Asignar números de radicado
  • Rechazar planes si es necesario
  • Cancelar radicaciones

3. Flujo Completo del Proceso

Flujo Principal (Camino Exitoso)
INICIO
Evaluación < 80 puntos

PROCESO DE FIRMAS
CS envía carta de evaluación

FIRMADO Y ENVIADO
Proveedor firma y envía plan

ESPERANDO APROBACIÓN
Técnico/Gestor revisa plan

EN RADICACIÓN
Plan aprobado, listo para radicar

PM RADICADO
Gestor Compras asigna número

PM REEVALUADO
App externa reevalúa proveedor

FIN
Caminos Alternativos
  • No Recibido (30 días): Si no se recibe el plan firmado en 30 días, pasa automáticamente a estado "No Recibido"
  • Aclaración: El técnico puede solicitar aclaración si el plan no es recibido
  • Solicitud de Ajustes: Si el plan requiere modificaciones, se solicitan ajustes al proveedor
  • Rechazo: El Gestor de Compras puede rechazar el plan durante la radicación
  • Falta de Ética: El Gestor puede marcar una falta de ética (suspensión de 5 años)

4. Guía para Proveedores

4.1 Acceder al Sistema
  1. Ingrese a la URL del sistema
  2. Use sus credenciales (usuario y contraseña proporcionadas por el Gestor)
  3. En el primer ingreso, se le pedirá cambiar la contraseña
4.2 Ver sus Evaluaciones
  1. En el dashboard principal, verá todas sus evaluaciones
  2. Las evaluaciones con puntaje < 80 requieren plan de mejoramiento
  3. Haga clic en "Ver Evaluación" para ver detalles
4.3 Crear un Plan de Mejoramiento
Solo puede crear un plan si tiene una evaluación que lo requiera.
  1. Haga clic en "Crear Plan de Mejoramiento"
  2. Complete los campos obligatorios:
    • Análisis de Causa Raíz: Identifique las causas del bajo desempeño
    • Acciones Propuestas: Describa qué hará para mejorar
    • Responsable: Nombre del responsable de implementar el plan
    • Fecha de Implementación: Cuándo se completará el plan
    • Indicadores de Seguimiento: Cómo medirá el éxito
  3. Agregue acciones de mejora individuales (opcional pero recomendado)
  4. Adjunte documentos de soporte (opcional)
  5. Haga clic en "Enviar Plan"
4.4 Firmar y Enviar Plan
  1. Recibirá la carta de evaluación de CS
  2. Descargue el plan generado
  3. Firme el plan físicamente o digitalmente
  4. En el sistema, vaya a su plan
  5. Haga clic en "Confirmar Envío del Plan Firmado"
  6. El plan pasará a estado "Firmado y Enviado"
4.5 Responder Aclaraciones

Si el sistema solicita aclaración:

  1. Verá un mensaje en el dashboard
  2. Abra el plan
  3. Lea las observaciones de aclaración
  4. Haga clic en "Responder Aclaración"
  5. Proporcione la información solicitada
4.6 Realizar Ajustes

Si se solicitan ajustes al plan:

  1. El plan pasará a estado "Requiere Ajustes"
  2. Verá los comentarios del técnico
  3. Haga clic en "Editar Plan"
  4. Realice las modificaciones solicitadas
  5. Guarde y envíe nuevamente

5. Guía para Técnico/Gestor

5.1 Crear Evaluaciones
  1. Vaya a "Crear Evaluación"
  2. Seleccione el proveedor
  3. Ingrese los puntajes por criterio:
    • Gestión
    • Calidad
    • Oportunidad
    • Ambiental y Social
    • Seguridad y Salud en el Trabajo (SST)
  4. El sistema calculará el puntaje total automáticamente
  5. Si el puntaje es < 80, se notificará que requiere plan
5.2 Enviar Carta de Evaluación
  1. Abra el plan del proveedor
  2. Haga clic en "Enviar Carta de Evaluación"
  3. Adjunte la carta firmada por CS (PDF o Word)
  4. Agregue comentario si es necesario
  5. El plan pasará a "Proceso de Firmas"
5.3 Confirmar Recepción de Plan
  1. Cuando el proveedor confirme el envío firmado
  2. Abra el plan
  3. Haga clic en "Confirmar Recepción"
  4. El plan pasará a "Esperando Aprobación"
5.4 Revisar y Aprobar Plan
  1. Revise el contenido del plan cuidadosamente
  2. Verifique acciones propuestas y cronograma
  3. Opciones disponibles:
    • Aprobar para Radicación: El plan cumple requisitos
    • Solicitar Ajustes: Requiere modificaciones menores
    • Rechazar: El plan no es aceptable (solo en radicación)
  4. Agregue comentarios explicativos
5.5 Solicitar Aclaración

Si un plan no es recibido después de 30 días:

  1. El sistema automáticamente marca el plan como "No Recibido"
  2. Vaya a "Planes No Recibidos"
  3. Seleccione el plan
  4. Haga clic en "Solicitar Aclaración"
  5. Describa qué necesita del proveedor
5.6 Marcar Falta de Ética (Solo Gestor)
Advertencia: Esta acción suspende al proveedor por 5 años y NO se puede deshacer.
  1. Solo disponible si hay evidencia de falta grave
  2. Abra el plan
  3. Haga clic en "Marcar Falta de Ética"
  4. Documente detalladamente el motivo
  5. Escriba "SUSPENDER" para confirmar
  6. Acepte el checkbox de confirmación

6. Guía para Gestor de Compras

6.1 Ver Planes Pendientes
  1. En el dashboard, haga clic en "Planes Pendientes de Radicación"
  2. Verá todos los planes en estado "En Radicación"
  3. La tabla muestra proveedor, puntaje, y revisor
6.2 Radicar un Plan
  1. Haga clic en "Radicar" en el plan deseado
  2. Ingrese el número de radicado según formato de su entidad
    Ejemplo: RAD-2025-001 o PM-2025-10-001
  3. Agregue comentario si es necesario
  4. Confirme la acción
  5. El sistema asignará el número y cambiará estado a "PM Radicado"
Importante: El número de radicado NO puede ser modificado después.
6.3 Rechazar un Plan
  1. Si un plan no cumple requisitos para radicación
  2. Haga clic en "Rechazar"
  3. Proporcione motivo detallado del rechazo
  4. Confirme la acción
  5. El plan pasará a "Rechazado"
6.4 Cancelar Radicación
  1. Si un plan rechazado debe ser cancelado oficialmente
  2. Abra el plan rechazado
  3. Haga clic en "Cancelar Radicación"
  4. El plan pasará a "Cancelación Radicada"

7. Estados del Plan

Estado Tipo Descripción Quién Actúa
BORRADOR Inicial Plan creado pero no enviado Proveedor
PROCESO_FIRMAS Principal Carta enviada, esperando firma del proveedor Proveedor
FIRMADO_ENVIADO Principal Proveedor confirmó envío del plan firmado Técnico/Gestor
ESPERANDO_APROBACION Principal Plan recibido, en revisión Técnico/Gestor
EN_RADICACION Principal Plan aprobado, listo para radicar Gestor Compras
PM_RADICADO Principal Plan con número de radicado asignado Sistema
PM_REEVALUADO Principal Proveedor reevaluado en app externa Sistema
NO_RECIBIDO Excepción Pasaron 30 días sin recibir plan Sistema (auto)
ACLARACION Excepción Se solicitó aclaración al proveedor Proveedor
SOLICITUD_AJUSTES Excepción Plan requiere ajustes menores Proveedor
RECHAZADO Excepción Plan no aceptado para radicación Gestor Compras
FALTA_ETICA Excepción Proveedor suspendido por 5 años N/A
FIN Final Proceso completado N/A

8. Acciones Disponibles por Estado

Vista Rápida de Acciones

  • BORRADOR: Editar plan, Enviar plan
  • PROCESO_FIRMAS: Confirmar envío del plan firmado
  • REQUIERE_AJUSTES: Editar y reenviar plan
  • ACLARACION: Responder aclaración
  • Cualquier estado: Ver historial

  • BORRADOR/ENVIADO: Enviar carta de evaluación
  • FIRMADO_ENVIADO: Confirmar recepción
  • NO_RECIBIDO: Solicitar aclaración
  • ESPERANDO_APROBACION: Aprobar para radicación, Solicitar ajustes
  • PROCESO_FIRMAS (Solo Gestor): Marcar falta de ética
  • Cualquier estado: Ver historial, Cambiar estado

  • EN_RADICACION: Radicar plan (asignar número), Rechazar plan
  • RECHAZADO: Cancelar radicación
  • PM_RADICADO: Ver número de radicado asignado
  • Cualquier estado: Ver historial

9. Preguntas Frecuentes (FAQ)

¿Qué pasa si no envío mi plan en 30 días?

El sistema automáticamente cambiará el estado a "No Recibido" y se solicitará una aclaración.

¿Puedo modificar un plan después de enviarlo?

Solo si se encuentra en estado "Requiere Ajustes" o "Borrador". Una vez aprobado, no se puede modificar.

¿Qué significa "Radicar" un plan?

Radicar significa asignar un número oficial de seguimiento en el sistema de gestión documental de la entidad.

¿Cuánto tiempo tengo para implementar mi plan?

La fecha de implementación la define usted en el formulario del plan. Se recomienda ser realista.

¿Qué pasa si marco incorrectamente una falta de ética?

Esta acción NO se puede deshacer. El proveedor quedará suspendido por 5 años. Use con extrema precaución.

¿Puedo ver el historial completo de un plan?

Sí, todos los usuarios pueden hacer clic en "Ver Historial" para ver todos los cambios de estado.

¿Qué formato debe tener el número de radicado?

Siga el formato establecido por su entidad. Ejemplos comunes: RAD-2025-001, PM-2025-10-001

¿Puedo adjuntar archivos al plan?

Sí, puede adjuntar documentos de soporte (PDF, Word, Excel) al crear o editar un plan.
